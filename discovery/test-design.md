# 🔮 Testory — 나의 전생 영혼동물 (테스트 설계)

**컨셉**: 세 가지 기운(빛·하늘·불꽃)이 깨어나며 결정되는 8종의 신화 아키타입 동물.
신비로운 전생 서사 위에 "어 나 진짜 이래…" 싶은 찐현실 디테일을 얹어, 캡처해서 단톡방에 던지고 싶게.

설계 방식: 3개의 이분 축(2³ = 8유형). 모든 유형 도달 가능, 점수 자동 밸런스.

---

## 3개의 축

| 축 | A극 | B극 |
|---|---|---|
| **기원의 빛** (어디서 빛나는가) | 광(光) — 드러내고 이끄는 양(陽)의 영혼 | 영(影) — 감추고 헤아리는 음(陰)의 영혼 |
| **영혼의 거처** (어디에 깃드는가) | 천(天) — 이상과 자유를 좇는 하늘의 영혼 | 지(地) — 뿌리와 현실을 지키는 땅의 영혼 |
| **기운의 흐름** (어떻게 움직이는가) | 화(火) — 단숨에 타오르는 불꽃의 영혼 | 수(水) — 천천히 스며드는 물결의 영혼 |

### 점수 로직 (전수 검증 완료)

- **1:1 매핑**: 8개 캐릭터 코드(`A1-B1-C1` ~ `A2-B2-C2`)는 3축 조합 2³=8가지와 정확히 1:1 대응. 8유형 전부 도달 가능.
- **동점 불가 (핵심)**: 각 보기는 한 축의 한쪽 극에만 +1. 축마다 7문항이라 두 극 점수 합이 **7(홀수)** → 3.5:3.5 동점이 수학적으로 불가능. 각 축은 7표 다수결로 승자 극이 항상 유일.
- **전수 결과**: 축별 2⁷=128 경로 시뮬레이션에서 동점 0건, 각 축 양극 모두 도달 → 8유형 전부 도달.
- **희귀도 합**: 봉황10 + 학12 + 백호10 + 늑대13 + 매11 + 용6 + 흑표범16 + 흰여우22 = **정확히 100%**.
